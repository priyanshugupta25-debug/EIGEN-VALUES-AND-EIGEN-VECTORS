Eigen Values and Eigen Vectors

Introduction

Eigen values and eigen vectors are important concepts in linear algebra. They are widely used in engineering, physics, computer science, and data analysis. They help in understanding linear transformations represented by matrices.

Definition

For a square matrix A, a non-zero vector X is called an Eigen Vector of A if:

AX = λX

where λ is a scalar known as the Eigen Value corresponding to the eigen vector X.

Steps to Find Eigen Values

  1. Form the identity matrix I of the same order as matrix A.
  2. Subtract λI from A to get (A − λI).
  3. Find the determinant of (A − λI).
  4. Set the determinant equal to zero:
    |A − λI| = 0
  5. Solve the resulting characteristic equation to obtain eigen values λ.

Steps to Find Eigen Vectors

  1. Substitute each eigen value λ into (A − λI).
  2. Solve the homogeneous system:
    (A − λI)X = 0
  3. Find the non-zero solution vector X.
  4. The vector X obtained is the eigen vector corresponding to eigen value λ.

Example

Let A = [ [2, 1], [1, 2] ]

Step 1: Form |A − λI|
|2−λ   1|
|1   2−λ|

Step 2: Solve determinant:
(2−λ)² − 1 = 0

Step 3: Eigen values are λ = 1, 3

Applications

  • Stability analysis in engineering systems
  • Image processing and computer vision
  • Quantum mechanics
  • Data science and machine learning
